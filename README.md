# CRUD API

## Description

This project is a simple CRUD (Create, Read, Update, Delete) API built using Node.js and TypeScript. It allows users to perform basic operations on data through API endpoints.

## Features

* Create new records
* Read existing records
* Update records
* Delete records
* Organized project structure

## Project Structure

* **src/** – source code of the application
* **dist/** – compiled JavaScript output
* **tests/** – testing files
* **node_modules/** – installed dependencies
* **package.json** – project dependencies and scripts
* **tsconfig.json** – TypeScript configuration

## Installation

1. Clone the repository
2. Install dependencies

```
npm install
```

## Run the Project

```
npm run dev
```

or

```
npm start
```
